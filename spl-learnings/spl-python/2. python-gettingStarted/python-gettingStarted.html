Python - Getting started

Python is much simpler to programme in. Any basic plain text editor, such as notepad or notepad++, can be used to create Python programs.


Create a simple "hello world" python program


    step 1: open any text editor like notepad++ or notepad.

    Step2: write below code
    	print("hello world")

    step3: save the file with any name but with .py extension like "helloWold.py"


Congratulation, you wrote one simplest python program which will display "hello world" once you execute this program.

Note: for executing any python program, python software must be installed on your system. so now you have to install the python software as per your system.


Python Installation for Windows PC


step1: Verify the Python Installation

Go to Start and enter 'cmd' in the search bar. Click Command Prompt.
Enter the following command in the command prompt:
python --version # checks the installed python version

or

Open window terminal(cmd.exe) and type "python", to check out if python is already installed and which version is installed.
If Python is already installed then you will get a message like below

    $ python
    Python 3.6.8 (default, Sep 10 2021, 09:13:53)
    [GCC 8.5.0 20210514 (Red Hat 8.5.0-3)] on linux
    Type "help", "copyright", "credits" or "license" for more information.
    >>>

if you don't see output like above then you need to download the python installer software from official website of Python https://www.python.org/ for windows pc. complete python installation docs can be found here https://docs.python.org/3/using/index.html


step2: Install Python

once python software downloaded, install it on your system.


step3: Adding Python to the Environment Variables (optional)

Skip this step if you have selected "Add Python to environment variables" during installation.

If you want to access Python through the command line but you didn’t add Python to your environment variables during installation, then you can still do it manually.

Before you start, locate the Python installation directory on your system.The following directories are examples of the default directory paths:

C:\Program Files\Pythonxv: if you selected "Install for all users" during installation, then the directory will be system wide.

C:\Users\Sammy\AppData\Local\Programs\Python\Pythonxv: if you didn’t select Install for all users during installation, then the directory will be in the Windows user path.

Path Configuration:


    1. Go to Start and enter advanced system settings in the search bar.
    2. Click View advanced system settings.
    3. In the System Properties dialog, click the Advanced tab and then click Environment Variables.
    4. Depending on your installation:
    4.1. If you selected Install for all users during installation, select Path from the list of System Variables and click Edit.
    4.2. If you didn’t select Install for all users during installation, select Path from the list of User Variables and click Edit.
    5. Click New and enter the Python directory path, then click OK until all the dialogs are closed.


step4: Verify the Python Installation

follow step1


Run The Python Script

Once python successfully Installed on your PC, now you are ready to execute any pyhton script

to run any python script just open the command prompt (cmd.exe) in your current working directory and type 'python scriptName.py'
