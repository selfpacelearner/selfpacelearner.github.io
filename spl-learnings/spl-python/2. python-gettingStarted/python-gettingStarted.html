Python - Getting started

Python is much simpler to programme in. Any basic plain text editor, such as notepad or notepad++, can be used to create Python programs.


Create a simple "hello world" python program


step 1: open any text editor like notepad++ or notepad.

Step2: write below code

	print("hello world")

step3: save the file with any name but with .py extension like "helloWold.py"


Congratulation, you wrote one simplest python program which will display "hello world" once you execute this program.

for executing any python program python software must be installed on your system. so now you have to install the python software as per your system.


Python Installation for Windows PC


step1: Open window terminal(cmd.exe) and type "python", to check out if python is already installed and which version is installed.
If Python is already installed then you will get a message like below

    $ python
    Python 3.6.8 (default, Sep 10 2021, 09:13:53)
    [GCC 8.5.0 20210514 (Red Hat 8.5.0-3)] on linux
    Type "help", "copyright", "credits" or "license" for more information.
    >>>

if you don't see output like above then you need to download the python software from official website of Python https://www.python.org/ for windows pc. complete python installation docs can be found here https://docs.python.org/3/using/index.html


step2: once python software downloaded, install it on your system.
